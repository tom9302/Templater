Introduction

Glowing Icons is a small collection of 16 icons with an optional glowing animation related to programming. It is very simple to use.

How to use

  • Make sure to include the external link. or the provided CSS file and fonts folder in the same folder with your HTML file.
  • Click on the icon you want to use.
  • Copy the HTML code and paste it in your HTML file.

Files

  • glowing-icons.html file :

    This is the file you should open to use the icons.

  • fonts folder

  • style.css file :

    This file contains all styles including the colors and animations of icons.
